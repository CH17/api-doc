Getting Started

{Project Title} RestFull API will provide JSON data based on passing parameters and autheticated by the access token.

API END POINT(s) Example: 

 Listings: http://localhost:8080/api/{v1}/listings

 Agents: http://localhost:8080/api/{v1}/agents

 Single Product: http://localhost:8080/api/{v1}/get-product/{id}

API Authetication

API must needs to be authenticated with the access token. A required parameter access_token must be valid.

Parameter Description Example Default Required
access_token Access Token for API. Can be generated from API dashboard. (32 Digits charectars) None YES

Get Listings

API END POINT TO GET LISTINGS: 

http://localhost:8080/{v1}/listings

API needs to have right parameters to get right data. Here is the list of available parameters.

Parameter Description Example Default Required
limit Limit of Data return by the API. Only integer number is allowed. 30 20 No
max_price Maximum price of the listings we need. Only integer number is allowed. 3000000 None No
min_price Minimum price of the listings we need. Only integer number is allowed. 1000000 None No
price Listing with that specific Price. Only integer number is allowed. 2500000 None No
city Name of the City to get listing of that specific City. Maple Ridge None No
state Name of the Province to get listing of that specific Province. British Columbia None No
address Address or part of address. 43 SUNSET ESTATES None No
bed Number of Bedrooms. Only integer number is allowed. 3 None No
bath Number of Bathrooms. Only integer number is allowed. 2 None No
zip_code Zip Code or PostalCode of any location to get data from that location. V0N1V0 None No
agent_id Get data from any specific Agent. Only integer number is allowed. 1432434 None No
max_sqft Maximum size of the listing in Sqft. 217800 None No
min_sqft Minium size of the listing in Sqft. 15800 None No
property_type Type of Listing. Multile types allowed separated by comma. Available Options:

sf = Single Family
mf = Multi Family
land = Vacant Land
condo = Condo
am = Appartment
com = Commercial
rent = Rental
bs = Business
ofc = Office

dayson The latest updated day for the listing. if 200, then property updated within last 200 days will retrun. 30 None No
lat Latitude of the center point. lat, lng, radius and unit parameters depends on each other. 59.46561 None No

Sorting Data

API allows to Sort data by specific Order. It will need two parameters. sort and order

Parameter Description Example Default Required
sort sort with specific field Like Price or Size. Available Options:

pricebed bathsizebuilt_date updated_at 		price ID No
order Sorting Order. Ascending and Descending is available.

asc = Ascending ( Sort returned data by ascending order )
desc = Descending ( Sort returned data by descending order )		 desc asc No
